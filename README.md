# Grafika
Egy textúrázott kockát jelenít meg a képernyőn, és lehetőséget ad a felhasználónak arra, hogy megváltoztassa a kocka színét a billentyűzet segítségével.
A kocka szinét egy billentyűgomb lenyomásával lehet majd változtatni, 3 szin lesz elérhető a piros a zöld és a kék. A kockánál egy ecsetes megoldás lesz majd, ahol a kurzor segitségével lehet festeni rá. A kamerakezelés a nyilak segitségével történne, a lenyomott nyilnak megfelelő irányba frissülne a pozició. A fények átalakitásra lesz lehetőség, a szinét és pozicióját is lehet majd változtatni egyaránt.
